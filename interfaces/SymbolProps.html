algebraic: boolean

commutative

commutative: boolean

A commutative expression. A commutative expression commutes with all other expressions under multiplication. If an expression a has commutative then a * b == b * a for any other expression b (even if b is not commutative). Unlike all other assumptions predicates commutative must always be true or false and can never be undefined. Also unlike all other predicates commutative defaults to true.
